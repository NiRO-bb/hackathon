"Хакатон по параллельному программированию на языке C#"

Из 13 команд-участников решение было предоставлено пятью комнадами. Я занял 4-ое место.

Для запуска проекта необходимо:
1. Открыть файл решения hackathon.sln в Visual Studio;
2. Добавить файл in.txt в папку "hackathon\bin\Debug\net9.0\";
3. Построить и запустить проект.
